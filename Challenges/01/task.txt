Хистограма

Напишете функция (histogram xs), която приема последователност от числа и връща вектор, в който на всяка
позиция с индекс n е съпоставен броя пъти, в които числото n се среща в подадената последователност.
Върнатия вектор трябва да има max + 1 елемента, където max е най-голямото число в подадената последователност.

(histogram [0 0 0 0 2 2]) ; [4 0 2]
(histogram '(1 1 1 4))    ; [0 3 0 0 1]
(histogram [0])           ; [1]

histogram не е дефинирана за празна последователност. В подадената последователност има само неотрицателни
цели числа.
